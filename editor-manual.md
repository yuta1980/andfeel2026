# and feel. 建築設計事務所 — 投稿マニュアル

このマニュアルでは、ウェブサイトのブログ記事（投稿）と作品紹介（かたち）の作成・編集・削除の手順を説明します。

---

## ログイン方法

1. ブラウザで `https://andfeel-architect.com/wp-admin/` にアクセスします
2. ユーザー名とパスワードを入力して「ログイン」をクリックします
3. ログインするとダッシュボード（管理画面のトップページ）が表示されます

ログイン後、左側のメニューに「ダッシュボード」「投稿」「かたち」の3つが表示されます。

---

## 第1章：ブログ記事（投稿）

ブログ記事は、日々の活動やお知らせなどを発信するための機能です。サイトの「日々」ページに一覧表示されます。

### 1-1. 記事の一覧を見る

左メニューの「投稿」をクリックすると、投稿一覧画面が表示されます。ここでは作成済みの記事がすべて表示され、タイトル・投稿者・カテゴリー・日付を確認できます。

### 1-2. 新しい記事を作成する

#### 手順

1. 左メニュー「投稿」にカーソルを合わせると、サブメニューが表示されます
2. 「投稿を追加」をクリックします（または投稿一覧画面の上部にある「投稿を追加」ボタンでもOK）
3. 以下の項目を入力します：

#### タイトルの入力

画面上部の「タイトルを追加」と表示されている欄に、記事のタイトルを入力します。

例：`現場見学会を開催しました`

#### 本文の入力

タイトルの下にあるエディター（テキスト編集エリア）に本文を入力します。エディターの上部にツールバーがあり、以下の書式設定が使えます：

| ボタン | 機能 |
|--------|------|
| **B** | 太字にする |
| *I* | 斜体（イタリック）にする |
| リストアイコン | 箇条書きリストを作成 |
| 数字リストアイコン | 番号付きリストを作成 |
| " （引用） | 引用ブロックを作成 |
| 揃えアイコン | テキストの左揃え・中央揃え・右揃え |
| リンクアイコン | テキストにリンクを追加 |

エディターの右上に「ビジュアル」と「コード」のタブがあります。通常は「ビジュアル」タブで入力してください。「コード」タブはHTMLを直接編集するためのものです。

#### 画像を本文に挿入する

1. 本文中の画像を入れたい位置にカーソルを置きます
2. エディター上部の「メディアを追加」ボタンをクリックします
3. メディアライブラリが開きます：
   - **新しい画像をアップロードする場合：** 「ファイルをアップロード」タブをクリックし、「ファイルを選択」をクリックしてパソコンから画像を選びます
   - **既にアップロード済みの画像を使う場合：** 「メディアライブラリ」タブから使いたい画像をクリックして選択します
4. 右側に表示される設定で、必要に応じて以下を入力します：
   - **代替テキスト：** 画像の説明（例：「完成した住宅の外観」）
   - **配置：** 「なし」「左」「中央」「右」から選択
   - **サイズ：** 「フルサイズ」推奨
5. 「投稿に挿入」ボタンをクリックします

#### アイキャッチ画像を設定する

アイキャッチ画像は、記事一覧で表示されるサムネイル画像です。

1. 画面を下にスクロールして「アイキャッチ画像」のエリアを探します
2. 「アイキャッチ画像を設定」のリンクをクリックします
3. メディアライブラリが開くので、画像をアップロードまたは選択します
4. 「アイキャッチ画像を設定」ボタンをクリックします

#### カテゴリーの設定（任意）

記事にカテゴリーを設定できます。画面を下にスクロールして「カテゴリー」エリアから該当するカテゴリーにチェックを入れてください。

### 1-3. 記事を保存・公開する

画面を下にスクロールすると「公開」のボックスがあります。

| ボタン | 説明 |
|--------|------|
| **下書き保存** | まだ公開せず、下書きとして保存します。後から編集を続けられます |
| **プレビュー** | 公開前に実際の表示を確認できます。新しいタブでプレビューが開きます |
| **公開** | 記事をサイトに公開します。公開後は誰でも閲覧できるようになります |

初めて公開する場合、「公開」ボタンをクリックすると記事が即座にサイトに反映されます。

### 1-4. 記事を編集する

1. 左メニュー「投稿」→「投稿一覧」をクリックします
2. 編集したい記事のタイトルにカーソルを合わせると、「編集 | クイック編集 | ゴミ箱 | 表示」のリンクが表示されます
3. 「編集」をクリックすると編集画面が開きます
4. 内容を修正し、「更新」ボタンをクリックして保存します

### 1-5. 記事を削除する

1. 投稿一覧で削除したい記事のタイトルにカーソルを合わせます
2. 表示されるリンクから「ゴミ箱」をクリックします
3. 記事がゴミ箱に移動します（すぐには完全削除されません）

## 第2章：作品紹介（かたち）

「かたち」は建築作品を紹介するための専用投稿タイプです。サイトの「かたち」ページに一覧表示されます。ブログ記事よりも入力項目が多く、ギャラリー写真や建物の詳細情報を登録できます。

### 2-1. 作品の一覧を見る

左メニューの「かたち」をクリックすると、作品一覧画面が表示されます。

### 2-2. 新しい作品を作成する

#### 手順

1. 左メニュー「かたち」にカーソルを合わせると、サブメニューが表示されます
2. 「新しい作品を追加」をクリックします

#### タイトルの入力

画面上部の「タイトルを追加」欄に、作品のタイトル（建物名や物件名）を入力します。

例：`徳島の家`、`街角のカフェ`

#### 本文の入力

ブログ記事と同様に、エディターに作品の説明文を入力します。建物のコンセプトや設計の意図など、作品の紹介文を書いてください。

#### 作品カテゴリーの設定

画面を下にスクロールして「作品カテゴリー」のエリアを探し、該当するカテゴリーにチェックを入れます。

| カテゴリー | 用途 |
|------------|------|
| **住宅** | 住宅の設計作品 |
| **店舗** | 店舗・商業施設の設計作品 |

カテゴリーは必ずどちらか1つ以上にチェックを入れてください。サイトの「かたち」一覧ページでフィルタリング（住宅/店舗の切り替え）に使われます。

#### アイキャッチ画像を設定する

アイキャッチ画像は、作品一覧で表示されるメイン写真です。作品を象徴する最も良い写真を選んでください。

1. 「アイキャッチ画像」エリアの「アイキャッチ画像を設定」をクリックします
2. 画像をアップロードまたは選択します
3. 「アイキャッチ画像を設定」ボタンをクリックします

#### ギャラリー写真を追加する（最大8枚）

ギャラリー写真は、作品の詳細ページで表示される追加の写真です。外観・内観・ディテールなど、複数のアングルの写真を登録できます。

1. 「ギャラリー写真（最大8枚・複数選択可）」エリアを探します
2. 「写真を追加」ボタンをクリックします
3. メディアライブラリが開きます
4. **複数枚をまとめて選択するには：** Ctrlキー（Macの場合はCommandキー）を押しながら写真をクリックしていきます
5. 選択が終わったら「投稿に挿入」または「ギャラリーに追加」ボタンをクリックします
6. 追加した写真はドラッグで並び替え可能です

#### 作品の詳細を入力する

「作品の詳細」エリアには、建物の基本情報を入力します。

| 項目 | 入力例 | 説明 |
|------|--------|------|
| **所在地** | `徳島県徳島市` | 建物の所在地（市区町村まででOK） |
| **竣工年** | `2024` | 建物が完成した年（西暦・数字のみ） |
| **構造** | `木造2階建` | 建物の構造種別 |
| **延床面積** | `120.50㎡` | 延床面積（単位含む） |

すべて入力が終わったら、画面上部にスクロールして「公開」ボタンをクリックしてください。

### 2-3. 作品を編集する

1. 左メニュー「かたち」→「すべての作品」をクリックします
2. 編集したい作品のタイトルにカーソルを合わせ、「編集」をクリックします
3. 内容を修正し、「更新」ボタンをクリックして保存します

### 2-4. 作品を削除する

1. 作品一覧で削除したい作品のタイトルにカーソルを合わせます
2. 「ゴミ箱」をクリックします

## 第3章：画像のアップロードについて

### 推奨する画像の設定

| 項目 | 推奨値 |
|------|--------|
| **ファイル形式** | JPG（写真の場合） |
| **横幅** | 最大 2048px |
| **ファイルサイズ** | 1枚あたり 2MB 以下が理想 |

画像が大きすぎるとサイトの表示速度が遅くなります。スマートフォンで撮影した写真はそのままでも大丈夫ですが、一眼レフなどの高解像度画像は事前にリサイズすることをおすすめします。

### アップロード方法

1. 記事や作品の編集画面で「メディアを追加」または「写真を追加」をクリック
2. 「ファイルをアップロード」タブを選択
3. 「ファイルを選択」をクリックしてパソコンから画像を選択、またはブラウザにドラッグ＆ドロップ
4. アップロードが完了したら、画像を選択して挿入します

## 第4章：よくある操作のヒント

### 下書きを後から編集するには

下書き保存した記事は、投稿一覧（またはかたち一覧）に「下書き」と表示されます。タイトルをクリックして編集を再開できます。

### 公開した記事を非公開にするには

1. 該当の記事を「編集」で開きます
2. 「公開」ボックス内の「ステータス」横の「編集」リンクをクリックします
3. ステータスを「下書き」に変更して「OK」をクリックします
4. 「更新」ボタンをクリックします

### 投稿のプレビュー

「プレビュー」ボタンをクリックすると、公開前の記事がどのように表示されるか確認できます。新しいタブでプレビューが開きます。内容を確認してから「公開」または「更新」してください。
